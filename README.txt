⚠️ ВАЖНО: Файлы ДОЛЖНЫ открываться через локальный веб-сервер, а не как обычный файл!

═══════════════════════════════════════════════
СПОСОБ 1 - САМЫЙ ПРОСТОЙ (Батник):
═══════════════════════════════════════════════

1. В этой папке найдите файл "start-server.bat"
2. Двойной клик по нему
3. Откроется окно консоли с адресом http://localhost:8000
4. Откройте браузер и перейдите по этому адресу
5. ВСЕ ФОТО И ВИДЕО ДОЛЖНЫ ПОЯВИТЬСЯ! ✨

═══════════════════════════════════════════════
СПОСОБ 2 - Если батник не работает (Python):
═══════════════════════════════════════════════

1. Убедитесь, что Python установлен (https://www.python.org/)
2. Откройте PowerShell/CMD в этой папке
3. Выполните команду:
   python -m http.server 8000
4. Перейдите на http://localhost:8000 в браузере

═══════════════════════════════════════════════
СПОСОБ 3 - VS Code (если установлен):
═══════════════════════════════════════════════

1. Откройте папку в VS Code
2. Установите расширение "Live Server"
3. Кликните правой кнопкой на index.html
4. Выберите "Open with Live Server"

═══════════════════════════════════════════════
ОБЯЗАТЕЛЬНО:
═══════════════════════════════════════════════

❌ НЕ открывайте файл двойным кликом!
❌ НЕ используйте file:// протокол!
✅ ДА используйте http://localhost:8000

═══════════════════════════════════════════════
